# 🛍️ ShopEase - Full Stack E-Commerce Marketplace

A modern full-stack e-commerce marketplace built with **Node.js**, **Express.js**, **MongoDB**, **EJS**, and **Tailwind CSS**. ShopEase allows users to browse products, request seller access, and manage products through a dedicated seller dashboard with an admin approval system.

---

## ✨ Features

### 👤 Authentication
- User Registration & Login
- Secure JWT Authentication
- Password Hashing using bcrypt
- Logout Functionality

### 🛒 Marketplace
- Browse Products
- Modern Product Cards
- Shopping Cart
- Dark & Light Mode
- Responsive UI

- Seller Dashboard

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/mohammadfaizamat-droid/shopease-marketplace.git
```

Move into the project folder

```bash
cd shopease-marketplace
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
MONGODB_URI=your_mongodb_connection
JWT_KEY=your_secret_key
EXPRESS_SESSION_SECRET=your_session_secret
```

Run the project

```bash
npm start
```

or

```bash
npm run dev
```
