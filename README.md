## рџљЂ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/shopease-marketplace.git
```

Move into the project folder

```bash
cd shopease-marketplace
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
MONGODB_URI=your_mongodb_connection
JWT_KEY=your_secret_key
EXPRESS_SESSION_SECRET=your_session_secret
```

Run the project

```bash
npm start
```

or

```bash
npm run dev
```
